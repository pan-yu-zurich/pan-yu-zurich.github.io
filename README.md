# pan-yu-zurich.github.io

Personal academic website.

## Editing

All content is in `index.html`. Edit directly and push to the `main` branch — GitHub Pages will redeploy automatically (usually within a minute).

Key places to edit:
- **News banner**: search for `Latest Publication`
- **Publications**: search for `const PAPERS` (the array of papers)
- **Working papers / Submitted**: search for `const WORKING`
- **Coauthor links**: search for `COAUTHOR_URLS`
- **About / bio**: search for `About —`
- **Teaching**: search for `const TEACHING`
